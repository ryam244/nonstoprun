アプリ企画書：『Non-Stop Run』
バージョン: 1.0
作成日: 2026年1月20日
作成者: Manus
1. 概要
『Non-Stop Run』は、信号待ちのストレスからランナーを解放し、「ノンストップで走る」という快適な体験を提供することに特化したiOS/Android向けランニングコース生成アプリである。
ユーザーが「走りたい距離」を入力するだけで、現在地を基点とした信号の少ない最適な周回コースを瞬時に自動生成する。
2. コンセプト
「最高のランニング体験は、最高のコースから始まる」
既存のランニングアプリが「走った後」の記録・共有に焦点を当てる中、本アプリは**「走る前」のコース設計という未開拓の領域に特化する。Googleマップのような「移動」目的の地図アプリではなく、「走ること自体（回遊）」を目的とする、全く新しい『ランニング専用カーナビ』**を目指す。
3. 解決する課題 (Pain Point)
信号待ちによるストレス: 信号でペースや心拍数が乱され、トレーニングの質が低下し、モチベーションが削がれる。
コース探しの手間: 毎回同じコースでは飽きてしまうが、信号が少なく、安全で、気持ちよく走れる新しいコースを自力で探すのは非常に面倒。
都市部での環境問題: 交通量の多い道を走ることによる、排気ガスや騒音への不快感。
4. ターゲットペルソナ
ペルソナ1：パフォーマンス向上志向ランナー (佐藤 健太, 32歳)
悩み: フルマラソンの目標達成のため、限られた時間で質の高いトレーニングをしたいが、市街地の信号待ちが最大の障害となっている。
提供価値: トレーニング効果を最大化する「必須ツール」。ペースを維持できるコースの提供により、目標達成を直接的にサポートする。
ペルソナ2：習慣化したい健康志向ランナー (鈴木 美咲, 28歳)
悩み: ランニングを始めたが、信号で気分が萎えてしまい、習慣化できない。楽しく気持ちよく走れるコースが知りたい。
提供価値: ランニングを楽しく継続させる「パーソナルコーチ」。走る楽しさを引き出し、習慣化を強力に後押しする。
5. 競合との差別化戦略
項目
Strava / Nike Run Club
Non-Stop Run
ポジショニング
ランニングSNS / 記録ツール
ランニング専用コースプランナー
主戦場
走った後の記録・分析・共有
走る前のコース設計・提案
コア機能
GPSログ、コミュニティ
信号回避ルート、周回コース自動生成
戦略
競合するのではなく**「共存」する。本アプリで作成したコースをGarmin等に転送し、記録はStrava等に共有できる連携機能**を提供。「Stravaと一緒に使う必須アプリ」の地位を確立する。
6. 主要機能とUI/UX
コアコンセプト
ワンタップに近い手軽さ: 起動後、距離入力とタップだけでコースを生成。
ランニング特化地図: 信号、緑道、高低差など、コースの「走りやすさ」を直感的に可視化。
画面設計
トップ画面:
中央に配置した距離入力ホイールで、走りたい距離を直感的に設定。
「ノンストップコースを探す」ボタンをタップするだけのシンプル設計。
コース提案画面:
特徴の異なる複数のコース（例：緑道優先、最短距離など）を地図上にプレビュー。
各コースの「信号数」「緑地率」「高低差」をアイコンやグラフで分かりやすく比較検討できる。
ナビゲーション画面:
シンプルな画面表示と、安全な走行をサポートする音声ナビゲーションが中心。
「Stravaで記録」「Garminに転送」ボタンを配置し、既存アプリへのスムーズな連携を実現。
7. 収益化モデル（サブスクリプション）
無料版:
基本的なコース生成機能（広告あり）。
コースの保存数や生成回数に制限を設ける。
有料版 (Proプラン / 月額500円程度):
広告非表示。
コース生成・保存が無制限。
「坂道を避ける」「夜間安全モード」などの高度な検索オプション。
高低差グラフや詳細なコース分析機能。
ウェアラブルデバイスへの高度な連携機能。
8. MVP（Minimum Viable Product）の定義
目的: コアバリューである「信号のないコースを手軽に作れる」ことがユーザーに受け入れられるかを最速で検証する。
必須機能:
現在地を基点とした距離入力機能。
信号情報を考慮した単一の周回コースを生成・表示する機能。
生成されたコースを地図上で確認できる機能。
含めない機能:
ユーザー登録、広告、有料プラン、ナビゲーション機能、外部アプリ連携、詳細なオプション設定。
9. 今後の展望（スケールアップ）
コミュニティ機能: ユーザー作成コースの共有、評価、ランキング機能。
BtoB展開: スポーツメーカーやジムとのタイアップ、マラソン大会への公式コース提供。
グローバル展開: 多言語対応による海外ランニング市場への進出。
